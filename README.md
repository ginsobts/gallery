# Gallery - 交互式 2D 画廊场景编辑器

一个基于 Unity 的 2D 俯视角交互式画廊场景系统，支持在运行时（Play Mode 和打包后）对场景中的元素进行创建、编辑和配置。

## 快速开始

1. 打开 Unity 项目，进入 `gallery_editor` 场景
2. 点击 Play 运行
3. 按 **Tab** 键打开/关闭编辑器

## 运行时编辑器使用方法

### 基础操作

| 按键 | 功能 |
|------|------|
| Tab | 开启/关闭编辑模式 |
| 鼠标点击 | 选中场景中的元素 |
| Esc | 取消当前操作 |

### 工具栏按钮

编辑模式下左上角显示工具栏：

- **鼠标 (选择/拖拽)** — 开启后可点击场景中任意元素进行选中和编辑
- **+ 照片** — 添加新的照片元素到场景
- **+ 视频** — 添加新的视频元素
- **+ NPC对话** — 添加对话型 NPC
- **+ NPC跟随** — 添加跟随型 NPC
- **+ 天气** — 添加天气区域
- **场景布局** — 打开场景布局设置（相机区块、过渡效果、时间轴）
- **当前区块设置** — 编辑当前所在区块的背景/灯光/天气/滤镜/BGM
- **保存** — 保存当前场景
- **场景列表** — 浏览和切换不同场景

### 元素编辑

选中元素后，元素周围会出现操作面板：

**上方按钮栏：**
- 选择文件 — 为元素选择媒体文件（图片/视频）
- 设置交互 — 配置元素的交互属性
- 复制 — 复制选中元素
- 顶层/底层 — 调整元素的显示层级
- 删除 — 删除选中元素

**缩放操作：**
- 选中元素后四周边缘和四个顶点会出现缩放手柄
- 拖拽左右边缘中点 — 横向缩放
- 拖拽上下边缘中点 — 纵向缩放
- 拖拽四个顶点 — 等比缩放
- 直接拖拽元素本身 — 移动位置

### 交通工具

- 左上角可切换交通工具（步行/自行车等），或按配置的快捷键切换
- 每种交通工具有独立的 Sprite、帧动画和移动速度
- 初始状态为步行，可在 Player prefab 的 GalleryVehicle 组件中配置可用交通工具

### 场景布局设置

点击"场景布局"按钮打开，可配置：

- **相机区块 — 区域分界线模式**
  1. 点击"创建区域分界线"按钮
  2. 面板关闭，场景中出现一条跟随鼠标的白色竖线
  3. 左键点击放置分界线（可连续放置多条）
  4. 右键或 Esc 结束放置
  5. 玩家穿过分界线时，相机自动切换到下一个区块
  - 分界线数量 = 区块数量 - 1
  - 支持"清除所有分界线"重新设置
- **过渡效果** — 配置不同属性在区块间的过渡方式（Cut/Fade/Lerp）
- **时间轴 — 交互式节点放置模式**
  1. 点击"放置时间轴节点"按钮
  2. 面板关闭，场景中出现跟随鼠标的预览圆点和连线
  3. 左键点击放置节点（可连续放置多个）
  4. 右键或 Esc 结束放置
  5. 点击"编辑时间轴节点"进入拖拽编辑模式（可拖动/删除已有节点）
  - 支持配置连线宽度、圆点大小、文字大小
  - 支持为每个节点设置日期文本
  - 支持"清除全部节点"重新设置

### 当前区块设置

点击"当前区块设置"按钮打开，自动编辑玩家当前所在区块：

- **背景** — 选择背景图片、调整缩放
- **灯光** — 环境亮度、颜色配置
- **天气** — 启用/配置天气特效（雨/雪/雾/阳光/萤火虫）
- **滤镜** — 颜色滤镜和风格化效果
- **BGM** — 选择背景音乐、设置音量和淡入时长

### 元素交互属性

在"设置交互"面板中可配置：

- **按键交互** — 玩家按指定键触发效果（缩放查看、显示文字、播放音效、切换BGM、改变天气/背景/亮度、跳转场景、开关物体）
- **靠近触发** — 玩家接近时自动触发效果，每个触发效果可单独配置"仅触发一次"或"可重复触发"
- **文字效果** — 显示文字时支持多种动画效果：打字机、渐隐、上滑、下滑、缩放、闪烁、波浪、故障风格
- 照片支持：碰撞体开关、天气特效附着、渐入效果、说明文字
- NPC 支持：对话内容编辑、跟随参数配置

### 地面绘制

场景中支持笔刷绘制地面纹理：
- 可自定义笔刷材质（在 Inspector 中配置）
- 支持调整笔刷大小
- 支持多种纹理平铺

## 数据存储

- 场景数据保存在 `Application.persistentDataPath/GalleryScenes/场景名/` 目录下
- 包含 `scene.json`（场景配置）和 `media/`（导入的媒体文件）
- 编辑器会在 Unity 停止运行后自动记住上次编辑的场景

## 打包说明

1. 确保 `gallery_editor` 场景已加入 Build Settings
2. 打包后所有编辑器功能正常可用
3. 导入文件功能使用系统原生文件选择器（Windows）

## Unity 编辑器集成

### 场景预览

在不运行游戏的情况下预览场景内容：
1. 选中 Hierarchy 中挂有 `RuntimeGalleryBootstrap` 的对象
2. Inspector 底部"场景预览"区域选择场景 → 点击"显示预览"
3. Scene 视图中会显示所有场景元素的位置和贴图

预览对象标记为 EditorOnly，不影响运行时行为和保存。

### 玩家出生点配置

在编辑器中（不进入 Play Mode）设置玩家出生位置：
1. 在 Bootstrap Inspector 中点击"创建出生点标记"
2. 在 Scene 视图中拖拽绿色的 Spawn Point 标记到目标位置
3. 点击"保存出生点到场景"将位置写入 scene.json

### Play 模式下拖拽

Play 模式下可以直接在 Scene 视图中拖拽 Player 和其他物体，位置会实时同步到 Game 视图。

## 项目结构

```
Assets/Scripts/Gallery/
├── Runtime/                    # 运行时编辑器系统
│   ├── RuntimeEditor.cs        # 编辑器主控制器
│   ├── RuntimeSceneBuilder.cs  # 场景构建/加载
│   ├── RuntimeElementHandle.cs # 元素操作手柄
│   ├── RuntimeSettingsPanel.cs # 元素属性面板
│   ├── RuntimeSceneSettingsPanel.cs  # 场景布局面板
│   ├── RuntimeBlockSettingsPanel.cs  # 区块设置面板
│   ├── BlockSettingsManager.cs # 区块设置运行时管理
│   ├── RuntimeAssetLoader.cs   # 资源加载缓存
│   ├── RuntimeUIHelper.cs      # UI 工具类
│   ├── SceneData.cs            # 数据结构定义
│   └── ...
├── GalleryPlayer.cs            # 玩家控制器
├── GalleryVehicle.cs           # 交通工具切换系统
├── GalleryCamera.cs            # 区块相机系统
├── GalleryFrame.cs             # 图片/交互系统
├── GalleryTextEffect.cs        # 文字动画效果
├── GalleryWeather.cs           # 天气特效系统
├── GalleryBackground.cs        # 背景渐变系统
├── GalleryTimeline.cs          # 时间轴系统
├── GallerySpawnPoint.cs        # 出生点编辑器标记
└── ...                         # 其他功能模块
Assets/Scripts/Editor/
├── GalleryScenePreview.cs      # 场景预览 + 出生点 Inspector
├── GallerySpawnPointEditor.cs  # 出生点 Scene 视图手柄
└── ...                         # 其他编辑器工具
```
