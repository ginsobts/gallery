# Gallery - 交互式 2D 画廊场景编辑器

一个基于 Unity 的 2D 俯视角交互式画廊场景系统，支持在运行时（Play Mode 和打包后）对场景中的元素进行创建、编辑和配置。

## 快速开始

1. 打开 Unity 项目，进入 `gallery_editor` 场景
2. 点击 Play 运行
3. 按 **Tab** 键打开/关闭编辑器

## 运行时编辑器使用方法

### 基础操作

| 按键 | 功能 |
|------|------|
| Tab | 开启/关闭编辑模式 |
| 鼠标点击 | 选中场景中的元素 |
| Esc | 取消当前操作 |

### 工具栏按钮

编辑模式下左上角显示工具栏：

- **鼠标 (选择/拖拽)** — 开启后可点击场景中任意元素进行选中和编辑
- **+ 照片** — 添加新的照片元素到场景
- **+ 视频** — 添加新的视频元素
- **+ NPC对话** — 添加对话型 NPC
- **+ NPC跟随** — 添加跟随型 NPC
- **+ 天气** — 添加天气区域
- **场景布局** — 打开场景布局设置（相机区块、过渡效果、时间轴）
- **当前区块设置** — 编辑当前所在区块的背景/灯光/天气/滤镜/BGM
- **保存** — 保存当前场景
- **场景列表** — 浏览和切换不同场景

### 元素编辑

选中元素后，元素周围会出现操作面板：

**上方按钮栏：**
- 选择文件 — 为元素选择媒体文件（图片/视频）
- 设置交互 — 配置元素的交互属性
- 复制 — 复制选中元素
- 顶层/底层 — 调整元素的显示层级
- 删除 — 删除选中元素

**下方操作栏：**
- 移动 — 拖拽此按钮移动元素位置
- 横向缩放 — 水平方向拉伸
- 纵向缩放 — 垂直方向拉伸
- 等比缩放 — 保持比例缩放

### 场景布局设置

点击"场景布局"按钮打开，可配置：

- **相机区块 — 区域分界线模式**
  1. 点击"创建区域分界线"按钮
  2. 面板关闭，场景中出现一条跟随鼠标的白色竖线
  3. 左键点击放置分界线（可连续放置多条）
  4. 右键或 Esc 结束放置
  5. 玩家穿过分界线时，相机自动切换到下一个区块
  - 分界线数量 = 区块数量 - 1
  - 支持"清除所有分界线"重新设置
- **过渡效果** — 配置不同属性在区块间的过渡方式（Cut/Fade/Lerp）
- **时间轴** — 添加/编辑时间轴节点，支持点击场景定位

### 当前区块设置

点击"当前区块设置"按钮打开，自动编辑玩家当前所在区块：

- **背景** — 选择背景图片、调整缩放
- **灯光** — 环境亮度、颜色配置
- **天气** — 启用/配置天气特效（雨/雪/雾/阳光/萤火虫）
- **滤镜** — 颜色滤镜和风格化效果
- **BGM** — 选择背景音乐、设置音量和淡入时长

### 元素交互属性

在"设置交互"面板中可配置：

- **按键交互** — 玩家按指定键触发效果（缩放查看、显示文字、播放音效、切换BGM、改变天气/背景/亮度、跳转场景、开关物体）
- **靠近触发** — 玩家接近时自动触发效果
- 照片支持：碰撞体开关、天气特效附着、渐入效果、说明文字
- NPC 支持：对话内容编辑、跟随参数配置

### 地面绘制

场景中支持笔刷绘制地面纹理：
- 可自定义笔刷材质（在 Inspector 中配置）
- 支持调整笔刷大小
- 支持多种纹理平铺

## 数据存储

- 场景数据保存在 `Application.persistentDataPath/GalleryScenes/场景名/` 目录下
- 包含 `scene.json`（场景配置）和 `media/`（导入的媒体文件）
- 编辑器会在 Unity 停止运行后自动记住上次编辑的场景

## 打包说明

1. 确保 `gallery_editor` 场景已加入 Build Settings
2. 打包后所有编辑器功能正常可用
3. 导入文件功能使用系统原生文件选择器（Windows）

## 项目结构

```
Assets/Scripts/Gallery/
├── Runtime/                    # 运行时编辑器系统
│   ├── RuntimeEditor.cs        # 编辑器主控制器
│   ├── RuntimeSceneBuilder.cs  # 场景构建/加载
│   ├── RuntimeElementHandle.cs # 元素操作手柄
│   ├── RuntimeSettingsPanel.cs # 元素属性面板
│   ├── RuntimeSceneSettingsPanel.cs  # 场景布局面板
│   ├── RuntimeBlockSettingsPanel.cs  # 区块设置面板
│   ├── BlockSettingsManager.cs # 区块设置运行时管理
│   ├── RuntimeAssetLoader.cs   # 资源加载缓存
│   ├── RuntimeUIHelper.cs      # UI 工具类
│   ├── SceneData.cs            # 数据结构定义
│   └── ...
├── GalleryPlayer.cs            # 玩家控制器
├── GalleryCamera.cs            # 区块相机系统
├── GalleryFrame.cs             # 图片/交互系统
├── GalleryWeather.cs           # 天气特效系统
├── GalleryBackground.cs        # 背景渐变系统
├── GalleryTimeline.cs          # 时间轴系统
└── ...                         # 其他功能模块
```
