# Gallery 场景完整配置指南

> 从零开始制作一个拥有图片、场景效果、背景、视频、NPC和镜头表现的完整场景。

---

## 一、基础流程概览

1. 打开 Unity，进入包含 `RuntimeGalleryBootstrap` 的编辑器场景（Gallery Editor）
2. 点击 Play 运行游戏
3. 按 **Tab** 键（或点击左上角编辑按钮）进入编辑模式——屏幕上下会出现灰色动态条纹
4. 进行所有配置操作
5. 按 **Tab** 或 **Esc** 退出编辑模式，自动保存

---

## 二、场景管理

**创建/切换场景：** 点击工具栏的「场景列表」按钮，可以：

- 查看所有已有场景
- 新建场景（输入名称）
- 切换到不同场景
- 删除不需要的场景

场景数据保存在 `persistentDataPath/Gallery/scenes/<场景名>/scene.json`，以及同目录下的媒体文件。

---

## 三、场景布局设置（「场景布局」按钮）

点击工具栏「场景布局」打开全局设置面板，包含以下部分：

### 3.1 玩家角色

| 操作 | 说明 |
|------|------|
| **选择角色贴图** | 选择一张图片作为玩家外观 |
| **帧率 (FPS)** | 动画播放速度，默认 6 |
| **行走帧（上/下/左/右）** | 每个方向的行走动画帧，支持多选图片文件 |
| **待机帧（上/下/左/右）** | 每个方向的待机动画帧 |

**省帧技巧：** 不需要提供全部8组帧。系统会自动回退——

- 只上传「行走-下」→ 所有方向都用这组帧
- 只上传「行走-左」和「行走-右」→ 缺失一侧时自动水平翻转另一侧
- 没有待机帧 → 自动取对应方向行走帧的第一帧作为静止画面

### 3.2 相机区块

相机区块系统用于控制 **镜头跟随和切换**，类似横版卷轴游戏的分区镜头。

| 操作 | 说明 |
|------|------|
| **过渡速度** | 镜头从一个区域滑到另一个区域的速度 |
| **相机 Y** | 镜头垂直位置 |
| **创建区域分界线** | 在场景中左键点击放置垂直分界线，把场景分为多个区块。右键结束 |
| **编辑区域分界线** | 拖拽菱形移动分界线位置，点击叉号删除 |

**工作原理：** 玩家向左走到分界线左侧，镜头自动滑动到左侧区块的中心；向右走过分界线，镜头滑到右侧区块中心。

**高级参数**（勾选「显示高级参数」）：

- 手动设置区块数量、起点X、区块宽度
- 点击场景设定相机起点
- 自动计算区块范围（根据场景元素分布）

### 3.3 过渡效果

当玩家在区块之间移动时，以下属性如何切换：

| 通道 | 选项 | 说明 |
|------|------|------|
| 背景 | Cut / Fade / Lerp | 背景图/颜色的切换方式 |
| 灯光 | Cut / Fade / Lerp | 环境光颜色和亮度 |
| 天气 | Cut / Fade / Lerp | 天气特效 |
| 滤镜 | Cut / Fade / Lerp | 画面色彩滤镜 |
| BGM | Cut / Fade / Lerp | 背景音乐 |

- **Cut** = 瞬间切换
- **Fade** = 淡入淡出
- **Lerp** = 平滑插值

还可设置全局 **过渡时长**（秒）。

### 3.4 时间轴

时间轴是场景中的一条可视化路径，用圆点和连线表示故事发展：

| 操作 | 说明 |
|------|------|
| **连线宽度 / 圆点大小 / 文字大小** | 控制时间轴外观 |
| **放置时间轴节点** | 进入放置模式，在场景中左键放圆点，右键结束 |
| **编辑时间轴节点** | 拖拽移动、删除节点 |
| **每个节点的日期文本** | 在节点上方显示文字标签 |

## 四、当前区块设置（「当前区块设置」按钮）

每个相机区块可以有 **独立的视觉风格和音效**。系统根据玩家当前所在区块自动应用。

### 4.1 背景

| 操作 | 说明 |
|------|------|
| **选择背景图片** | 每个区块可以有独立的背景 |
| **背景缩放 X/Y** | 调整背景图大小 |

### 4.2 灯光

| 参数 | 说明 |
|------|------|
| **环境亮度** | 0~1，控制整体明暗 |
| **环境色** | RGB颜色，影响全局照明色调 |
| **底色** | 场景默认底色 |

### 4.3 天气

| 参数 | 说明 |
|------|------|
| **启用天气** | 是否在该区块显示天气 |
| **天气类型** | 雨 / 雪 / 雾 / 阳光 / 萤火虫 |
| **粒子数量** | 天气效果密度 |
| **粒子颜色** | 天气粒子的颜色 |

### 4.4 画面滤镜

| 参数 | 说明 |
|------|------|
| **颜色滤镜** | 画面色调调整 |
| **滤镜强度** | 0~1 |
| **艺术风格** | 画面风格化 |
| **风格强度** | 0~1 |

### 4.5 音乐

| 参数 | 说明 |
|------|------|
| **选择BGM** | 背景音乐文件 |
| **音量** | 播放音量 |
| **淡入时间** | BGM切换时的淡入秒数 |

> **提示：** 有多个区块时可以用「从区块N复制」快速复制设置。

## 五、添加场景元素

在编辑模式工具栏中，点击对应按钮添加元素：

### 5.1 照片（+ 照片）

放置一张静态图片到场景中。

**基础设置：**

- 更换文件（选择图片）
- 位置 X/Y、缩放 X/Y
- 排序层（控制前后关系，数字越大越靠前）
- 碰撞体（是否阻挡玩家移动）
- 说明文字

**照片特有设置：**

- **靠近渐入** — 玩家靠近时图片逐渐显现
- **渐入距离 / 速度** — 控制渐入效果
- **附带天气特效** — 可在照片区域叠加独立天气（如飘雪、萤火虫）

### 5.2 视频（+ 视频）

放置可播放的视频到场景中。

**视频设置：**

- 自动播放 / 循环播放
- 播放键 — 指定哪个按键触发播放
- 触发范围 — 玩家靠多近才能触发
- 启用音频 / 最大音量 / 音频范围 — 空间化音效，越远越小声
- 靠近渐入
- 封面图 — 视频未播放时显示的静态画面

### 5.3 NPC 对话（+ NPC对话）

放置一个会说话的NPC。

**对话设置：**

- **自动触发** — 玩家靠近自动开始说话（否则需按键）
- **循环对话** — 对话说完后是否重复
- **触发距离 / 对话键**
- **打字速度 / 文字大小**
- **气泡颜色 / 文字颜色**
- **对话行** — 添加多行对话，每行可设置：
  - 文字内容
  - 停留时长
  - 文字效果：打字机 / 渐隐 / 上滑 / 下滑 / 缩放 / 闪烁 / 波浪 / 故障

**待机方向动画：**

- 帧率 + 上/下/左/右待机帧
- NPC会自动面向靠近的玩家

### 5.4 NPC 跟随（+ NPC跟随）

放置一个会跟随玩家的NPC。玩家触碰后开始跟随。

**跟随设置：**

- 跟随距离 / 跟随速度 / 记录间隔
- 动画帧率

**方向动画：**

- 行走帧（上/下/左/右）
- 待机帧（上/下/左/右）
- 跟随过程中自动根据移动方向播放对应方向动画

### 5.5 天气（+ 天气）

放置一个独立的天气效果区域。

**天气设置（与区块天气不同，这是一个可自由放置的局部天气区域）：**

- **天气类型** — 雨 / 雪 / 雾 / 阳光 / 萤火虫
- **粒子数量 / 强度**
- **粒子颜色**
- 位置 X/Y — 天气区域中心
- 范围 X/Y — 天气区域大小（通过拖拽缩放手柄或数值调整）

> **注意：** 天气元素没有点击交互，缩放改变的是天气范围而非粒子外观。

## 六、交互效果系统

**照片、视频、NPC对话** 三种元素可以配置丰富的交互效果，分为两种触发方式：

### 6.1 按键交互

玩家靠近后按指定键触发，可多次触发。

- **交互按键** — 指定按键
- **交互距离** — 触发范围

### 6.2 靠近触发

玩家进入范围自动触发。

- **触发距离**
- **仅触发一次** — 关闭后每次离开再靠近都会重新触发

### 6.3 可用的效果

两种触发方式均可配置以下效果（可同时勾选多个）：

| 效果 | 说明 |
|------|------|
| **缩放查看** | 全屏放大查看图片（按Esc/点击关闭） |
| **显示文字** | 弹出文字，可选8种文字效果 |
| **播放音效** | 选择音效文件 + 音量 |
| **切换BGM** | 替换背景音乐 + 音量 |
| **改变天气** | 全局天气类型/粒子数/颜色 |
| **改变背景** | 改变背景颜色 + 淡入时间 |
| **改变亮度** | 修改全局环境亮度 |
| **加载场景** | 跳转到其他Gallery场景 |
| **开关物体** | 通过元素ID显示/隐藏其他元素 |

## 七、元素操作（鼠标模式下）

进入编辑模式后，点击「鼠标 (选择/拖拽)」切换到鼠标模式：

| 操作 | 说明 |
|------|------|
| **左键点击** | 选中元素，出现蓝色边框和缩放手柄 |
| **拖拽元素** | 移动位置 |
| **拖拽边缘中点** | 单轴缩放 |
| **拖拽四角** | 等比缩放 |
| **Delete键** | 删除选中元素 |
| **点击空白处** | 取消选中 |

选中元素后可在右侧面板修改详细设置。

## 八、完整制作流程示例

假设你要做一个「四季花园」场景：

1. **创建场景** → 场景列表 → 新建「四季花园」
2. **设置玩家** → 场景布局 → 上传角色贴图和四方向行走/待机帧
3. **分区** → 创建3条分界线，把场景分成春/夏/秋/冬4个区块
4. **区块风格** → 分别进入每个区块设置：
   - 春：浅绿底色，萤火虫天气，欢快BGM
   - 夏：暖黄底色，阳光天气，蝉鸣BGM
   - 秋：橙红底色，雾天气，安静BGM
   - 冬：浅蓝底色，雪天气，温馨BGM
5. **放照片** → 每个区域放几张对应季节的照片，设置靠近渐入
6. **放NPC** → 春天区域放一个对话NPC，靠近后说"欢迎来到花园"
7. **加跟随者** → 秋天区域放一个跟随NPC，触碰后跟着玩家走
8. **配交互** → 给某张照片添加按键交互：按E缩放查看 + 显示说明文字
9. **加天气细节** → 在某张照片旁放一个小范围的萤火虫天气效果
10. **做时间轴** → 放几个时间轴节点标注"春→夏→秋→冬"
11. **设过渡** → 背景Fade、灯光Lerp、天气Fade，过渡时长1.5秒
12. **退出编辑** → 按Tab/Esc，自动保存

## 九、分发给其他人

使用 Unity 打包（Build）时，系统会自动将当前所有场景数据从 `persistentDataPath` 复制到 `StreamingAssets`（通过 `SceneDataBuildProcessor` 自动执行）。收到打包文件的用户首次运行时，会自动安装你预置的场景数据。新版本打包会通过版本号覆盖旧数据。

## 十、快捷键速查

| 快捷键 | 作用 |
|--------|------|
| **Tab** | 进入/退出编辑模式 |
| **Esc** | 退出编辑模式 / 退出放置模式 / 关闭缩放查看 |
| **WASD** | 玩家移动（非编辑模式） |
| **Delete** | 删除选中元素 |
| **左键** | 选中/拖拽元素；放置分界线/时间轴节点 |
| **右键** | 结束放置模式 |
